Functions

bind (Path)
Binds a table to a config directory. The returned table maps keys to configurations, which are handled by different "finders". Finders are handlers that try loading a config entry in a certain format and are tried sequentially until one succeeds. If no finder matches, nil is returned.

Parameters:

  • Path dir to the directory to look in.

Returns:

    table config A table that maps to the config directory

Usage:

    local main_config = config.bind 'configurations'
main_config.foo.bar
-- Loads some file like foo.json or foo.yaml
-- in the configurations directory
